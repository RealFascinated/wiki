# EXIF Data Viewer

The EXIF Data Viewer tool helps you inspect the metadata embedded in your images. EXIF (Exchangeable Image File Format) data contains valuable information about how and when a photo was taken.

## What You Can View

The tool displays various types of EXIF data, including:

- **Camera Information**
  - Make and model
  - Lens information
  - Software used

- **Capture Details**
  - Date and time
  - Exposure settings
  - Focal length
  - ISO sensitivity
  - Aperture

- **Location Data**
  - GPS coordinates
  - Altitude

## How to Use

1. Upload an image using the file picker below
2. Click "Read EXIF Data" to analyze the image
3. View the extracted metadata in an organized format

## Privacy Considerations

All processing is done in your browser. Your images are never uploaded to any server, ensuring complete privacy.

## Technical Details

The tool uses the `exifr` library to extract EXIF data. It supports:
- JPEG/JPG
- PNG
- WebP
- TIFF
- HEIC/HEIF

## Common Uses

- **Photography Analysis**: Learn about the camera settings used in photos
- **Privacy Check**: Verify what metadata is embedded in your images
- **Technical Learning**: Understand how different cameras record information
- **Content Verification**: Check the authenticity of images
